# 在线学习平台

一个功能完整的在线学习平台基础版本，支持课程浏览、课程报名和学习进度追踪。

## 功能特性

### 1. 课程列表
- 展示全部课程信息（封面、标题、讲师、简介、分类）
- 清晰的卡片式布局，便于浏览
- 已报名课程显示专属标识

### 2. 课程详情
- 完整课程介绍
- 课程章节信息
- 讲师介绍
- 适合人群说明

### 3. 课程报名
- 一键报名操作
- 报名成功弹窗提示
- 报名后自动显示"已报名"状态

### 4. 学习进度
- 已完成百分比展示
- 已学习章节数统计
- 个人学习区域管理

### 5. 页面状态
- 加载状态（loading spinner）
- 空状态（无课程/无报名时友好提示）
- 报名后状态切换

## 技术栈

- **前端**：HTML5 + CSS3 + JavaScript（原生，无框架依赖）
- **数据**：localStorage 本地持久化
- **设计**：响应式布局，支持桌面端

```
multica-2/
```

## 快速开始

1. 直接在浏览器中打开 `index.html` 文件
2. 浏览课程列表，点击课程卡片进入详情页
3. 点击"立即报名"完成课程报名
4. 在"我的学习"页面查看已报名课程及进度

## 课程数据

平台当前包含 6 门模拟课程，涵盖：
- 编程开发（Python 基础、Web 前端）
- 数据科学（数据分析与可视化）
- 产品运营（产品经理入门）
- 设计创意（UI/UX 设计基础）
- 人工智能（人工智能导论）

## 数据说明

当前版本使用 localStorage 存储用户数据：
- `enrolledCourses`：已报名课程 ID 列表
- `progressData`：各课程学习进度

如需接入真实后端，修改 `app.js` 中的数据请求逻辑即可。
